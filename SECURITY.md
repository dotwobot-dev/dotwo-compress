# Security

This is a small desktop application project and does not currently provide a
formal security response SLA.

Please do not publish exploit details, private media, or sensitive user data in
public issues. If GitHub security advisories are enabled for the repository,
use a private security advisory. Otherwise, contact the maintainers through the
private channel already known to the project.

Do not include API keys, access tokens, private media, or personal files in bug
reports or pull requests.
